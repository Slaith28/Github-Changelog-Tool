GitHub Changelog Tool is an AI-powered web app that transforms your commit history into a clean, structured changelog. 
  
Sign in with your GitHub, pick a repo, and get a formatted changelog grouped by release tag or date, categorized by commit type, with contributor attribution and direct
links to each diff. Export as `.md`, `.txt`, or `.json`.

Built as a personal project to explore agentic coding workflows with Claude Code.
